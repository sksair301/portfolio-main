## About This Project

This project is a personal portfolio website built with React, TypeScript, Vite, and Three.js. It is more than a static resume — it is an interactive experience that reflects my style as a developer and designer.

### What the website includes

- A polished landing experience with animated sections
- An interactive 3D character experience powered by Three.js
- Dedicated sections for:
  - About me
  - What I do
  - My career journey
  - Selected work
  - Technical stack
  - Contact information
- Smooth scrolling and motion effects for a modern feel
- Responsive design for desktop and mobile experiences

### Key technologies used

- React + TypeScript
- Vite
- React Router
- Three.js and React Three Fiber
- GSAP for animations
- CSS modules and custom styling
- Node.js/Express backend for contact-related functionality

## Project Structure

The app is organized into reusable components and sections, including:

- Landing page and navigation
- About and career sections
- Work showcase and tech stack sections
- Character/3D experience integration
- Contact and call-to-action sections

## Getting Started

### Prerequisites

Make sure you have the following installed:

- Node.js
- npm or yarn

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/Sunil-Yadav-1912/Portfolio-main.git
   cd Portfolio-main
   ```

2. Install dependencies
   ```bash
   npm install
   ```

3. Start the development server
   ```bash
   npm run dev
   ```

4. Open the local app in your browser

## Scripts

- npm run dev — starts the Vite app and local server
- npm run build — builds the project for production
- npm run preview — previews the production build
- npm run lint — runs the linter
